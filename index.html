How to use the timer

Enter the time you would like to use for each rest period.

Once you have finished doing your set click the Start Rest button.

The timer should begin counting down from the time you input, once the timer is complete your set count will increase to help you keep track of what set you are on in your workout.

Once you've finished all sets for the workout you're on click the Reset Set Count button.

You can adjust the amount of time for your rests anytime the timer is not running.
